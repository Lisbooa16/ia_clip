Publicação em TikTok, Instagram e YouTube

Guia rápido para postar os clips gerados em todas as plataformas de uma só vez.

✅ Caminho recomendado

Fluxo básico de publicação

1. Criar integrações oficiais (OAuth) para cada rede.
2. Salvar tokens + refresh em um cofre seguro (ex.: Vault ou banco com criptografia).
3. Enfileirar um job de publicação por plataforma.
4. Monitorar status e registrar URLs finais dos posts.
YouTube

Upload oficial

  • Use a YouTube Data API com o escopo youtube.upload.
  • Configure canal e playlist padrão para Shorts.
  • Envie título, descrição e tags programaticamente.
Instagram

Reels via Graph API

  • Conta precisa ser Business/Creator e conectada ao Meta.
  • Use o endpoint content publishing para Reels.
  • Requer aprovação e limites de taxa.
TikTok

Content Posting API

  • Necessita aprovação da TikTok for Developers.
  • Use o fluxo de upload + publish para vídeos curtos.
  • Mapeie categorias e legendas antes de publicar.

Como habilitar “postar tudo de uma vez”

  • Crie um formulário para selecionar plataformas e preencher título/legenda/hashtags.
  • Ao clicar em “Publicar”, dispare um job por plataforma em paralelo.
  • Exiba na tela o status (enfileirado, enviando, publicado, erro).
  • Salve o link final de cada post no histórico do job.
⚠️ Cada rede social exige autenticação oficial e aprovação. Não é permitido fazer upload por scraping. Use APIs oficiais para manter sua conta segura e evitar bloqueios.
